# 说明

用了1个月时间，把《Java编程思想》读了一遍。然后用Mindjet Mindmanager做了读书笔记，跟大家分享一下~

1. 这个思维导图的读书笔记没有涵盖书中的所有知识点，但是作为框架和读书的思路比较清晰。
2. 思维导图中有很多代码在note中，没有贴到博客中。
3. 思维导图中没有第3章、第22章《图形化用户界面》的总结，因为这个用得很少。
